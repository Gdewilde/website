Welcome to the Stargate community!

This project was open sourced in September 2020 and thus we have a fairly young but active community. We're here to explore new ways to interact with data at scale and have some fun along the way. We build in the open and use the channels below for communication.

Discussion: Discord

  • #general = use this channel for general discussion, ask questions, help others
  • #new-ideas = use this channel to discuss new features you'd like to see in the project
  • #contribute = use this channel to discuss project development and releases

News and Announcements on Twitter: @stargateio

Issue Tracking

Use the GitHub issue tracker for the various Stargate repositories to file bugs and feature requests. If you need help, you can find us on Discord in the #general (for bugs) and #new-ideas (for new features) channels.

Contributing

Nothing makes us happier than community contributions. See the README in the main stargate repository to get a grasp of the project layout and reference the CONTRIBUTING.md file in the respective repository for detailed steps. Documentation contributions are also appreciated and you can find instructions for the docs in the stargate/docs repository. If you need more guidance or have questions, please ask on Discord.

Code of Conduct

To make Stargate a welcoming and harassment-free experience for everyone, we adopt and follow the Contributor Covenant. Please read our Code of Conduct before engaging with the community.
